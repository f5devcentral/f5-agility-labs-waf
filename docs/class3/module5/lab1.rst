Blocking Mode
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

#. Navigate to Security>>Application Security>>Policy Building>>Learning and Blocking Settings

   .. image:: /_static/class3/image20.png
     :width: 5.34167in
     :height: 3.23542in

#. Change Enforcement mode to "Blocking"

#. Click "Apply Policy"
